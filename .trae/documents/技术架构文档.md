## 3. 页面布局

### 3.1 登录页 (/login)
- 居中登录卡片
- 两个登录入口：客户账号登录、运营人员登录
- 登录后根据账号类型跳转不同页面

### 3.2 运营后台 (/admin/*)
- 项目列表页 (/admin/upload)
- 项目详情页 (/admin/project/:id)
- 数据上传页 (/admin/project/:id/upload)

### 3.3 客户看板 (/dashboard)
- 顶部导航栏
- 数据展示区域（品牌排名、查询趋势等）
- GEO效果总结区域

| 字段名 | 类型 | 说明 |
|--------|------|------|
| id | integer | 主键 |

## 6. 核心功能说明

### 6.1 CSV/Excel 解析

使用 **papaparse** 库进行CSV解析，支持：
- Excel单元格内的换行符
- 包含逗号的字段
- 各种复杂格式

关键字段映射：
| 原始字段 | 映射字段 |
|----------|----------|
| 查询时间 | query_time |
| 序号 | id |
| Query：xxx | query |
| 位置1品牌 | brand1 |
| 位置2品牌 | brand2 |
| 位置3/4/5 | brand3/brand4/brand5 |
| 品牌问一问输出内容 | ai_content |
| GEO效果总结 | geo_summary |

### 6.2 登录逻辑

- **客户登录**：验证 projects 表中是否存在该邮箱的记录
- **运营登录**：通过后设置 localStorage.isAdmin = true

### 6.3 数据展示

Dashboard 通过 query_time 筛选不同日期的数据，支持：
- 品牌排名统计
- 查询趋势分析
- GEO效果总结展示

### 6.4 项目逻辑删除

- 项目删除为逻辑删除，仅修改 status 字段为 'deleted'
- 列表页默认不显示已删除项目
- 可通过"已删除"筛选器查看已删除项目
- 已删除项目可恢复为"已立项"状态

## 9. 常见问题排查

### 9.1 上传CSV后数据不显示
1. 检查Console日志中 `PapaParse - 映射后数据:` 是否包含 geo_summary 字段
2. 确认CSV表头是否包含 "GEO效果总结"
3. 检查日期格式是否被正确解析

### 9.2 项目删除失败
1. 检查Console错误信息
2. 确认Supabase数据库projects表的UPDATE权限
3. 确认status字段CHECK约束是否包含'deleted'

### 9.3 日期筛选不正常
1. 确认query_time字段有值
2. 检查数据格式是否为有效日期
