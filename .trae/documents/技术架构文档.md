## 3. 页面布局

### 3.1 登录页 (/login)
- 居中登录卡片
- 两个登录入口：客户账号登录、运营人员登录
- 登录后根据账号类型跳转不同页面

### 3.2 运营后台 (/admin/*)
- 项目列表页 (/admin/upload)
- 项目详情页 (/admin/project/:id)
- 数据上传页 (/admin/project/:id/upload)

### 3.3 客户看板 (/dashboard)
- 顶部导航栏
- 数据展示区域（品牌排名、查询趋势等）
- GEO效果总结区域

---

## 4. 路由配置

```typescript
Routes:
  /                       -> 重定向到 /login
  /login                  -> Login
  /project/apply          -> Onboarding (客户申请)
  /admin/upload           -> Upload (运营项目列表)
  /admin/project/:id      -> ProjectDetail (项目详情)
  /admin/project/:id/upload -> DataUpload (数据上传)
  /dashboard              -> Dashboard (客户看板)
```

## 5. 数据库表结构

### 5.1 projects 表
| 字段名 | 类型 | 说明 |
|--------|------|------|
| id | integer | 主键 |
| brand_name | text | 品牌名称 |
| product_name | text | 产品名称 |
| industry | text | 行业 |
| status | text | pending/approved/deleted |
| form_data | jsonb | 表单数据 |
| user_email | text | 客户邮箱 |
| created_at | timestamp | 创建时间 |
| updated_at | timestamp | 更新时间 |

**status 字段说明：**
- `pending` - 待审核
- `approved` - 已立项/已审核
- `deleted` - 已删除（逻辑删除）

### 5.2 project_data 表
| 字段名 | 类型 | 说明 |
|--------|------|------|
| id | integer | 主键 |
| project_id | integer | 外键关联projects |
| file_name | text | 文件名 |
| data | jsonb | Excel解析数据 |
| created_at | timestamp | 创建时间 |

## 6. 核心功能说明

### 6.1 CSV/Excel 解析

使用 **papaparse** 库进行CSV解析，支持：
- Excel单元格内的换行符
- 包含逗号的字段
- 各种复杂格式

关键字段映射：
| 原始字段 | 映射字段 |
|----------|----------|
| 查询时间 | query_time |
| 序号 | id |
| Query：xxx | query |
| 位置1品牌 | brand1 |
| 位置2品牌 | brand2 |
| 位置3/4/5 | brand3/brand4/brand5 |
| 品牌问一问输出内容 | ai_content |
| GEO效果总结 | geo_summary |

### 6.2 登录逻辑

- **客户登录**：验证 projects 表中是否存在该邮箱的记录
- **运营登录**：通过后设置 localStorage.isAdmin = true

### 6.3 数据展示

Dashboard 通过 query_time 筛选不同日期的数据，支持：
- 品牌排名统计
- 查询趋势分析
- GEO效果总结展示
