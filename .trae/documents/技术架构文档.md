# 小红书 GEO 服务监控工具 - 技术架构文档

## 1. 技术栈

| 层级 | 技术选型 |
|------|----------|
| 前端框架 | React 18 + TypeScript |
| 后端框架 | Express.js |
| 构建工具 | Vite |
| 样式方案 | Tailwind CSS |
| 状态管理 | Zustand |
| 路由 | React Router DOM v6 |
| 部署环境 | 海外阿里云服务器 |

---

## 2. 项目结构

```
Red/
├── src/                    # 前端代码
│   ├── components/         # 公共组件
│   ├── pages/              # 页面组件
│   │   ├── Login.tsx       # 登录页
│   │   ├── admin/         # 运营后台
│   │   │   ├── Onboarding.tsx    # 售前表单
│   │   │   └── Upload.tsx        # 数据上传
│   │   └── Dashboard.tsx  # 客户看板
│   ├── hooks/             # 自定义 Hooks
│   ├── utils/             # 工具函数
│   ├── App.tsx            # 应用入口
│   └── main.tsx           # 入口文件
├── api/                   # 后端代码
│   ├── routes/            # 路由
│   ├── controllers/       # 控制器
│   └── index.ts           # 服务入口
├── supabase/              # Supabase 配置
├── package.json           # 项目依赖
└── vite.config.ts         # Vite 配置
```

---

## 3. 页面布局

### 3.1 登录页 (/login)
- 居中登录卡片
- 用户名、密码输入框
- 登录按钮

### 3.2 运营后台 (/admin/*)
- 左侧导航栏
- 顶部 Header（显示当前用户）
- 主内容区域

### 3.3 客户看板 (/dashboard)
- 顶部导航栏
- 数据展示区域
- Top 3 Query 推荐卡片

---

## 4. 路由配置

```typescript
Routes:
  /login              -> Login
  /admin/onboarding  -> Onboarding (运营)
  /admin/upload      -> Upload (运营)
  /dashboard         -> Dashboard (客户)
```
